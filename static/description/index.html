Odoo Module: Project Stage Visibility

Technical Description

This module provides the capability to hide or show project stages based on user. It extends the base 'project' module in Odoo and adds a new model 'project.stage.visibility' with fields for users, stages, and a boolean field to determine visibility. It also extends the 'project.project' and 'project.project.stage' models to override the search method and adjust the visible stages based on the rules defined in the 'project.stage.visibility' model.

How to Use

1. Installation

Install this module as you would any other Odoo module: place it in the addons directory, update the app list in your Odoo instance, and then install it from the app list.

2. Configuration

After installation, navigate to Project -> Configuration -> Project Stage Visibility. Here, you can define visibility rules.

Each rule has the following fields:

  • Name: Enter the name of the rule for reference.
  • Users: Select the users to which this rule applies. The visibility of the selected stages will be modified for these users.
  • Stages: Select the stages to which this rule applies. These stages' visibility will be modified for the selected users.
  • Hide Stages: Check this if you want the selected stages to be hidden from the selected users. If unchecked, the stages will be visible.

3. Usage

Once the visibility rules are configured, the selected users will see the changes when they navigate to their projects. The stages that are set to be hidden will not be visible to them.
